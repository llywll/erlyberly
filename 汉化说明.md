## 汉化原则

1. **保持技术术语准确性**：如 "Cookie"、"Erlang"、"Elixir"、"LFE" 等专业术语保留原文
2. **界面简洁明了**：翻译后的文本长度适中，适合界面显示
3. **注释准确传达原意**：代码注释的汉化注重准确表达原英文注释的含义
4. **统一术语翻译**：相同的术语在整个项目中保持一致的翻译

## 编译和运行

汉化工作不会影响项目的编译和运行，因为：
- FXML 中的文本只是字符串值的改变
- Java 注释的汉化不影响代码逻辑
- 所有功能代码保持不变

### 验证步骤

```bash
# 清理并重新编译
mvn clean compile

# 运行应用程序
mvn javafx:run
```

## 注意事项

1. **编码问题**：确保源文件使用 UTF-8 编码，以正确显示中文字符
2. **字体支持**：JavaFX 界面需要系统支持中文字体显示
3. **IDE 设置**：在 IntelliJ IDEA 中确保文件编码设置为 UTF-8

## 新增功能

### 进程字典查看功能（2026-06-04）

已添加查看 Erlang 进程字典的功能：

**修改的文件：**
1. **erlyberly.erl** - 添加了 `get_process_dictionary/1` 函数
2. **NodeAPI.java** - 添加了 `getProcessDictionary()` 方法
3. **ProcController.java** - 添加了 `processDictionary()` 方法和 `ProcessDictionaryThread` 线程类
4. **ProcView.java** - 在进程列表右键菜单中添加了“查看进程字典”菜单项

**功能说明：**
- 在进程列表中右键点击任意进程
- 选择“查看进程字典”菜单项
- 系统会在新窗口中显示该进程的字典数据（键值对）
- 如果进程没有字典数据，会显示提示信息
